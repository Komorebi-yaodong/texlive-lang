\begin{abstract}
This document present list of serbian words, which contains >>fi<< and >>fl<< 
ligatures, in latin scripts, command for disable ligatures and output.
Two new commands which call \verb|\spfi| and \verb|\spfl| represent space 
betwen letter \textit{f} and \textit{i} in >>fi<< ligatures and space
betwen letter \textit{f} and \textit{l} in >>fl<< ligatures, retrospectrivly.
To this command you may control space betwen letters in ligatures. 
For this package this commands is represents in words that begin with lower case ž.
Others words contains \verb|\kern| command for space factor.
Space for >>fi<< ligatures is set to 0.03em and for >>fl<< ligatures to 0.08em.
For the better typography letters `ij' and `nj' is provide whith comand \verb|\ij| 
and \verb|\nj| and space bettwen this letters is set to -0.03em.
\end{abstract}

\textcolor{blue}{blue} contains >>fi<< ligatures and exist word which contain >>fl<<

\textcolor{magenta}{magenta} contains >>fi<< ligatures but not exist word which 
contains >>fl<< ligatures.
