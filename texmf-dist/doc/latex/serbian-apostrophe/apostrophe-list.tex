\begin{abstract}
This document present list of serbian words with apostrophe, in latin scripts, 
command for this words and output. If you don't now how is typing same word with 
apostrophe just typing a command. For the beter typography letters `ij', `nj', 
`kj', `aj', `zj' and `oj' is provide whith comand \verb|\ij|, \verb|\nj|, \verb|\kj|, 
\verb|\aj|, \verb|\zj| and \verb|\oj| and space bettwen this letters is set to -0.03em,
-0.03em, -0.02em, -0.01em, -0.02em and -0.02em retrospectrivly. 
\end{abstract}

\part{Some words with accent marks}

Serbian language has four accent characters, each of which is marked with the sign. 
These are the accents: short falling (fast) or in serbian: \textit{kratkosilazni (brzi)} 
which is marked -\textdoublegrave{a}-, a brief upward (slow) or in serbian: 
\textit{kratkouzlazni (spori)} which is marked -\`{a}-, the long downward (descending) or 
in serbian: \textit{dugosilazni (silazni)} which is marked -\^{a}- and a long upward 
(ascending) or in serbian: \textit{dugouzlazni (uzlazni)} which is marked -\'{a}-.
Genitive sign is placed on the final vowel genitive pluralwhere necessary
to indicate the difference of this form than others, usually the same genitive 
singular pronouns, sing as -\^{a}-.
